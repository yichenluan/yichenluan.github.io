寒假在用 SAE + Python + Tornado + Bootstrap 做网站，对于用户注册这块需要用到MD5加密，所以找了一些信息，记录如下。

- ###`hashlib`模块

	```
	import hashlib
	m = hashlib.md5()
	m.update('password')
	m.hexdigest()
	```
	
	(2)是将m初始化为md5 hash对象
	
	(3)是通过update方法对字符串就行md5加密
	
	(4)是输出加密后的16进制结果
	
- ###`md5`模块

	```
	import md5
	l = md5.new('password').hexdigest()
	```
